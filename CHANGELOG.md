# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog and this project adheres to Semantic Versioning.

## [1.0.0] - 2026-06-30

### Added

* Multi-threaded TCP port scanning
* Custom port range scanning
* Common ports scanning mode
* Host discovery functionality
* Service detection and identification
* Banner grabbing support
* HTTP title extraction
* CIDR range expansion and scanning
* JSON report generation
* HTML report generation
* TXT report generation
* Configurable timeout settings
* Verbose output mode
* Basic OS fingerprinting capabilities
* CVE reference hints for detected services
* Windows console color output

### Documentation

* Added project README
* Added MIT License
* Added release documentation

### Security

* Included usage disclaimer for authorized environments only
